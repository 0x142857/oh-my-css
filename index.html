Get started

Current version: 0.1.3 (Unstable)

Install

load Oh-My-CSS in your project

<!-- Oh My CSS core stylesheet -->
<link rel="stylesheet" href="//cdn.ohmycss.com/latest/css/ohmycss.css">

<!-- Oh My CSS lite version (without normalize.css) -->
<link rel="stylesheet" href="//cdn.ohmycss.com/latest/css/ohmycss.lite.css">

or Bower

bower install ohmycss --save

Components

Buttons

Do wanna bigger or smaller button? try the markup: 

<button class="oh-button button-$SIZE">button text</button> // $SIZE in ['default','large','small','mini']

Custom button color? use class oh-button button-colorful COLOR to customize

We now support:

white white-bg
gray gray-bg 
light-purple light-purple-bg 
yellow yellow-bg
black black-bg
blue blue-bg
green green-bg

Button group

HTML markup:
<div class="button-group">
  <button class="oh-button button-default white-bg">Left</div>
  <button class="oh-button button-default white-bg">Middle</div>
  <button class="oh-button button-default white-bg">Addon</div>
  <button class="oh-button button-default white-bg">Right</div>
</div>

Navbar

  </div>
</nav>
